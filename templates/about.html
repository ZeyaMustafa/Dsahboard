About This Dashboard

This supermarket sales dashboard provides interactive visualizations and analytics for your sales data.

Features

  • Date Range Selection: Filter data by custom date ranges or use preset options (Today, Yesterday, Last 7 Days, Last 30 Days, This Month, Last Month, Year-to-Date)
  • Interactive Charts: Built with Plotly for hover details, zoom, and pan capabilities
  • Data Export: Download filtered data as CSV
  • Sales Forecasting: 7-day moving average forecast for sales and profit
  • Business Intelligence: Top products analysis, category breakdown, and trend analysis

How to Use

  1. Use the date range controls at the top to select your desired period
  2. Explore the interactive charts - hover for details, click legend items to toggle visibility
  3. Use the Export CSV button to download the currently filtered data
  4. Navigate to Daily, Weekly, or Monthly views for quick period selection

Data Requirements

Place your CSV files in the data/ directory with the following columns:

  • Date (YYYY-MM-DD format)
  • Product Category
  • Sales Amount
  • Quantity Sold

The system will automatically calculate derived metrics like Total Sale and Total Profit.

Technical Details

Built with:

  • Flask (Python web framework)
  • Bootstrap 5 (responsive design)
  • Plotly (interactive charts)
  • Pandas & NumPy (data processing)
  • Matplotlib & Seaborn (static plotting fallback)
